    DisplayManager¶

    O DisplayManager é o componente central da biblioteca EasySSD1306. Ele gerencia todos os componentes visuais e coordena a renderização do display.

    Visão Geral¶

    O DisplayManager é responsável por: - Gerenciar o display SSD1306 - Coordenar a renderização de componentes - Controlar a navegação - Gerenciar estados da interface

    Inicialização¶

    DisplayManager display;
display.setDisplay(new Adafruit_SSD1306(128, 64));

    Configuração de Botões¶

    display.setButtons(BUTTON_UP_PIN, BUTTON_DOWN_PIN);

    Adicionando Componentes¶

    TextView text(1, "Texto de exemplo");
display.addComponent(&text);

    Renderização¶

    void loop() {
    display.render();
    delay(100);
}

    Estados¶

    O DisplayManager mantém controle de dois estados importantes: - currentState: Estado atual da interface - lastState: Estado anterior da interface

    Para mudar o estado: 

    display.setState(NOVO_ESTADO);

    Variáveis Estáticas¶

    • selectedItem: Índice do item selecionado
    • confirmItem: Flag de confirmação
    • cancelItem: Flag de cancelamento

    Exemplo Completo¶

    #include <EasySSD1306.h>

DisplayManager display;
TextView text(1, "Menu Principal");
Navigation menu(2);

void setup() {
    display.setDisplay(new Adafruit_SSD1306(128, 64));
    display.setButtons(2, 3);  // Pinos 2 e 3 para botões

    display.addComponent(&text);
    display.addComponent(&menu);
}

void loop() {
    display.render();
    delay(100);
}

    Dicas de Uso¶

    1. Inicialização
    2. Sempre inicialize o display antes de adicionar componentes

    3. Configure os botões se for usar navegação

    4. Renderização

    5. Chame render() no loop principal

    6. Mantenha um delay adequado para evitar flickering

    7. Estados

    8. Use estados para gerenciar diferentes telas

    9. Mantenha controle do estado anterior para navegação

    10. Componentes

    11. Adicione componentes na ordem desejada
    12. Cada componente deve ter um ID único
